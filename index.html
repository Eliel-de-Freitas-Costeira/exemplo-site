Curso Git e GitHub do Curso em Video

Sobre o Curso

O Curso é lecionado pelo professor Gustavo Guanabara no canal do YouTube Curso em Video com oferecimento da Hostnet.

O Curso tem 13 aula (foram feitos em forma de live mas que ficaram gravadas no canal do YouTube) e aborda os seguintes temas:

  • O que é Git? O que é versionamento?
  • O que é GitHub? Pra que ele serve?
  • A evolução do Git e GitHub
  • Instalações e configurações importantes
  • Criando o primeiro repositório
  • Instalando GitHub Desktop no LINUX
  • Clonando um repositório
  • Versionando seus projetos antigos
  • Você sabe usar Issues?
  • Guia da Linguagem Markdown
  • Seu GitHub mais seguro
  • Git Branches de forma fácil e com exemplo
  • Hospedagem grátis no GitHub Pages
